# 
- Java Development Kit (JDK): Version 11 or higher (JDK 17 or 21 is recommended).
- Apache Maven: Ensure mvn is accessible from your terminal/command prompt.
- ```Install Playwright Browsers using below command```
- ```mvn exec:java -e -D exec.mainClass=com.microsoft.playwright.CLI -D exec.args="install"```
- To Run test
- ```mvc clean and then mvn test -Dtest=CaseKaroTest```
